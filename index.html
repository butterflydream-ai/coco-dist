What it does

A small window that does a lot

No Electron. Native Swift and AppKit, about two megabytes, open the instant you ask for it.

App launcher

Fuzzy search that understands pinyin, romaji and Hangul initials, with actions on every result.

Clipboard history

Searchable history and favorites. Press Space to preview, then paste back into the app you came from.

Calculator

Watch an expression resolve as you type — and a running tape keeps past answers a keystroke away.

Units & currency

Convert units — and add money across currencies in one line, $100 + ¥800 + €20, at live rates with the trend.

Emoji & web

Find an emoji by name, or send the query to your search engine.

Shell aliases

Good to know

Questions

Can it add different currencies in one line?

Yes — type one line like $100 + ¥800 + €20 and Coco sums it at live rates, shown across the currencies you watch, each with its trend. More →

Does the calculator just give one answer?

No. It resolves as you type and keeps a running tape; the last answer continues the next line, like a desk calculator. More →

Can it find an app if I don't type the exact name?

Yes — fuzzy search that also understands pinyin, romaji and Hangul initials, so you type the way you think. More →

Does it remember what I copied?

Yes — a searchable clipboard history with favorites. Press Space to preview, then paste back where you were. More →

What else lives in the same box?

Unit conversion, emoji by name, a quick web search, and your own shell aliases — one hotkey, nothing to launch.
